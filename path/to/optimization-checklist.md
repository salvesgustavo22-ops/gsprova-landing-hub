# Otimizações para Imagens

1. **Formatos modernos**: Use WebP sempre que possível.
2. **Tamanhos adequados**: Ajustar as dimensões das imagens.
3. **Lazy loading**: Implementar loading assíncrono para imagens fora da viewport.
4. **Comprimir**: Utilize ferramentas de compressão para reduzir o tamanho dos arquivos sem perda significativa de qualidade.
